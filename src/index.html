    About this project

    This project aims to provide details on different Formula One season results (currently covering 2020-2023).

    This was made for COMP 3612—a web course provided at Mount Royal University. Thanks to Prof. Connolly for providing this assignment.

    Developer Information

    Project Developer: Richard Johnston <techpowerawaits@outlook.com>

    Project Repository: https://github.com/TechPowerAwaits/3612-Assign2

    Technologies Used

    • Tailwind CSS is used for stylizing the website.
    • Prettier is used for stylizing the HTML and JavaScript.
    • Tailwind CSS Plugin for Prettier sorts the Tailwind classes used.
    • Visual Studio Code is used for writing the HTML and JavaScript.
    • Tailwind CSS IntelliSense is used to provide Tailwind CSS suggestions while coding.
    • Prettier Formatter for Visual Studio Code allows the usage of Prettier within Visual Studio Code.
    • npm is used to manage the installation of Tailwind CSS and Prettier.
    • GitHub is used to store all the files and host the webpage.
