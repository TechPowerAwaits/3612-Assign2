About this project

This project aims to provide details on different Formula One season results (currently covering 2020-2023).

This was made for COMP 3612—a web course provided at Mount Royal University. Thanks to Prof. Connolly for providing this assignment.

Project Developer: Richard Johnston <techpowerawaits@outlook.com>

Project Repository: https://github.com/TechPowerAwaits/3612-Assign2

Technologies Used

  • Tailwind CSS—Used for stylizing the website.
  • Prettier—Used for stylizing the HTML and JavaScript.
  • Tailwind CSS Plugin for Prettier—Sorts the Tailwind classes used.
  • Visual Studio Code—Used for writing the HTML and JavaScript.
  • Tailwind CSS IntelliSense—Used to provide Tailwind CSS suggestions while coding.
  • Prettier Formatter for Visual Studio Code—Allows the use of Prettier within Visual Studio Code.
  • npm—Used to manage the installation of Tailwind CSS and Prettier.
  • GitHub—Used to store all the files.
